```markdown
## 📖 关于项目

**图词（VisionLex）** 是一个基于 AI 视觉识别技术的英语单词学习平台。

通过上传图片，AI 智能分析图片内容并识别出最能描述图片的英文单词（A1-A2级别），
提供详细解释、例句和语音朗读功能，让英语学习变得更加直观有趣。

### ✨ 核心特性

- 🖼️ **图片智能识别** - 基于 Moonshot AI 强大的视觉理解能力
- 📝 **单词学习** - 提供 A1-A2 级别适合学习的单词推荐
- 🔊 **语音朗读** - TTS 技术提供标准发音
- 💾 **学习记录** - 自动保存学习历史，支持收藏功能
- 📱 **响应式设计** - 完美适配桌面端和移动端
- 🎨 **现代化UI** - 蓝粉渐变主题，毛玻璃效果
```
